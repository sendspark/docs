Sendspark provides shortcuts to help you interact with the Chrome Extension without clicking buttons.

## **Sendspark Desktop App Shortcuts:**

- Space - Skip countdown
- ⌥+⌘+Q - Exit
- ⌥+⌘+X - Logout

## **Sendspark Chrome Extension Shortcuts:**

- ⇧+X - Toggle spotlight
- While recording (recording toolbar options):
  - ⇧+⌥+X - Delete current recording
  - ⇧+⌥+D - Restart current recording
  - ⇧+⌥+R - Pause current recording
  - ⇧+⌥+S - Stop current recording
