If your Sendspark recording won't upload or the extension stops responding, reload the extension: click the Extensions icon in Chrome, find the Sendspark extension, click **Reload**, and try recording again. This resets the connection and updates you to the latest version, resolving most connection-related issues. If it persists, restart Chrome and try once more.

If you’re experiencing recording problems — such as **“video upload failed”** — try **reloading your Chrome extension**. This usually resolves most connection-related issues.

**Why this works:**

- Resets the extension connection
- Fixes temporary glitches
- Ensures you’re running the **latest available version**

**Steps:**

1. Click the **Extensions** icon in Chrome.
2. Find the **Sendspark extension**.
3. Click **Reload**.
4. Try recording again.

If the issue continues, restart Chrome and try once more.

## Key takeaways

- Most recording and upload problems, including "video upload failed", are connection-related and clear up after reloading the extension.
- Reloading resets the extension connection, fixes temporary glitches, and ensures you're on the latest available version.
- To reload: Extensions icon → find Sendspark → **Reload** → try recording again.
- If it still doesn't work, restart Chrome and try once more.
