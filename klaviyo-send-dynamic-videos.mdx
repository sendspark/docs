## What is a Dynamic Video?

Dynamic videos are videos that are **automatically personalized**for each of your recipients.

With 1 single recording, you can make it look like you're recording multiple videos. The feature automatically adds each of your prospects' websites to the background and with the help of AI you can also add the prospect's name

This tutorial will cover **how to share dynamic videos in Klaviyo.**

If you are not familiar with dynamic videos, please start with this tutorial: [How to make dynamic videos](/how-to-create-a-dynamic-video-campaign))

## How to Share Dynamic Videos in Klaviyo

## Step 1: Copy the Video Snippet Code

1. Access your Dynamic Video Campaign.
2. Head to the SHARE CAMPAIGN button
3. Copy the snippet code provided by clicking **Copy to Clipboard** on Klaviyo

---

## Step 2: Download the CSV File

1. Export the CSV file containing the leads you generated videos for.
2. Save the file to your device for later upload into Klaviyo.

---

## Step 3: Upload Leads into Klaviyo

1. Log in to your Klaviyo account.
2. Navigate to **Audience &gt; Lists & Segments**.
3. Create a new list by selecting **Create List**, and name it (e.g., "Test").
4. Click **Upload Contacts** and select your CSV file.
5. Map the fields (e.g., First Name, Email, Video Link).
6. Click **Next**, ensure the contacts are set to "Subscribe," and click **Import**.

---

## Step 4: Create an Email Campaign

1. Go to **Campaigns** and select **Create Campaign**.
2. Name the campaign and choose **Email** as the type.
3. Select the list you uploaded (e.g., "Test").
4. Choose a template or create a new email layout.
5. In the email editor, drag and drop an **HTML block** into the email body.
6. Paste the snippet code you copied from Sendspark into the HTML block.

---

## Step 5: Preview and Test the Campaign

1. Use the preview function to ensure the video populates correctly in the email.
2. Test the campaign by sending it to a sample recipient or using Klaviyo’s preview tools.

---

## Optional: Set Up SMS Campaigns

If sending an SMS campaign:

1. Select **SMS** instead of **Email** during campaign creation.
2. Use the **Video Link** field from your CSV file as a variable in the SMS text body.
3. Note that HTML snippet codes are not compatible with SMS messages. The correct option here is to use the variables instead of the Snippet Code.
