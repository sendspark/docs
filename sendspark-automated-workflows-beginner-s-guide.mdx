Sendspark has an Automated Workflow feature to help you create personalized & send videos automatically, based on triggers. You can think of these as your own AI Video Sales Agents... Totally customizable, and totally automated!

Here's how they work.

## How Sendspark Automated Workflows Work

## How to Create Automated Workflows

You can create automated workflows in the [automated workflow](https://sendspark.com/automated-workflows) section of Sendspark.

Simply click **"New Automation"** to create a new automation from scratch.

Alternatively, you can click **"Create from Template"**to create an automated workflow from a template.

![New automation and Create from template buttons in Sendspark](/images/11172985/613ac1cc-image.png)

## How Triggers and Actions Work

In your automated workflow, you will see the option to add a trigger & action. This is how to think about them.

![New Automation editor with Add a trigger and Add an action rows](/images/11172985/aab93adc-image.png)

- **Trigger:**an event or criteria that "triggers" an automation to start. Common triggers include:
  - Form filled
  - Demo booked
  - Webpage visited
  - Lead status met
  - Event occurred
- **Action:**an "action" that you want to happen automatically after the trigger starts the campaign. For example, this could be to...
  - Automatically generate a Sendspark video
  - Automatically add a lead to an email campaign in Smartlead, Instantly, etc
  - Automatically send an email in Gmail or Outlook

Each automation should have one Trigger, and at least one action. There is no limit to the number of actions you can have... So go wild!

## How to Connect Automated Workflows with Dynamic Video Campaigns

You can connect Automated Workflows with Sendspark Dynamic Video campaigns to automatically create videos, or to automatically share videos after they've been created.

In either case, you would set "Sendspark" as the trigger of the action.

### Use Sendspark to Trigger Automations

To use Sendspark to trigger automations, select "Sendspark" as the trigger, and then choose which event you would like to use:

![Set up a trigger modal with Sendspark selected and event dropdown open](/images/11172985/55dd69be-image.png)

There are 3 main categories:

- **New Video Generated**— This fires when new dynamic videos are generated. Use this when creating automations that automatically send videos in email or LinkedIn after videos are generated.
- **New Video CTA Clicked**— This fires when end-viewers click on the call-to-action button embedded next to your video.
- **New Video Watched** — This fires when someone watches your video. Use this trigger when you want to create notifications when viewers engage with videos.

Additionally, with each event, you can choose whether each event fires for "all campaigns" or "Specific campaigns"

- **All Campaigns**— means that events from ANY dynamic video campaign will trigger this automation. This is best to use when creating notifications.
- **Specific Campaigns**— means that only one linked dynamic video campaign will trigger this automation. This is best to use when creating automations that automatically send videos after they are created.

### Use Sendspark as an Action in Automations

There is only one Sendspark automated action at the moment — to create dynamic videos!

![Add an action modal with Sendspark and Create dynamic video selected](/images/11172985/9d7b8d7e-image.png)

You can use this action to create a dynamic video when the trigger you've set occurs.

When you create dynamic videos via automations, you will be able to use data from the videos in subsequent actions.

![Email Body field with dropdown of Embed Url Share Url and Name values](/images/11172985/b9c3bf2c-image.png)

For example, among other data fields, you can use...

- **Share Url**— is the link to the video on a public share page
- **Email Embed HTML** — is a little snippet you can use to embed your video link + GIF in an email
- **Embed Url**— is a hosted video link you can use to embed your video on a landing page

This can help you share the video after it's been created. For example, you can automatically embed the video & send in Gmail or Outlook after it's been created.
