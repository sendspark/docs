To connect Expandi with Sendspark, generate your API credentials in Sendspark (Workspace ID, an API Key, and a Secret Key), then paste them into Expandi under Account Settings → Integrations Apps → the API Sendspark section. Once you save, both accounts are synced and you can send AI-personalized videos in your LinkedIn campaigns.

## Key takeaways

- You connect the two tools with three Sendspark credentials: your Workspace ID, an API Key, and a Secret Key.
- Find them in Sendspark under Settings → API Credentials.
- In Expandi, paste them under Account Settings → Integrations Apps → the API Sendspark section.
- This is a one-time setup. Afterward you can create and send AI-personalized videos in Expandi campaigns.

## This article will help you prepare your Expandi and Sendspark automated connection. Follow these steps to properly sync both accounts.

## 1. Get Your API Credentials from Sendspark

Access your API section in Sendspark: [https://sendspark.com/settings/api-credentials](https://sendspark.com/settings/api-credentials):

![API menu](/images/10223502/0c1bc3de-image.png)

Copy the Workspace ID and Create an API Key:

![Creating a New Key](/images/10223502/5f40c540-image.png)

Now, create a Secret Key:

![Creating a Secret Key](/images/10223502/92a8a51a-image.png)

## 2. Add API Credentials to Expandi

Head to your menu button located in the top right-hand side corner and click on "Account Settings":

![Expandi Menu](/images/10223502/c89e8c81-image.png)

To the left-hand side you'll find the tab called "Integrations Apps":

![Integrations App](/images/10223502/cc52d50c-image.png)

Now scroll down until you find the API Sendspark section and paste the information gathered from Sendspark:

![Sendspark Integration section in Expandi](/images/10223502/0ae4d1f4-image.png)

## 3. Apply Changes

That's it! You're all set!

Ready to send hyper-personalized LinkedIn Messages? Check out this article for how to [create & send videos in Expandi campaigns.](/expandi-integration-ai-personalized-videos)
