If your camera bubble (your face) is not showing when recording over a PDF, this is due to a Chrome security setting. By default, Chrome blocks extensions from accessing files (like PDFs opened from your computer), which prevents the camera overlay from appearing.

⚙️ **Enable access to PDF files**

1️⃣ Open extension settings

Click the 🧩 puzzle icon (top right in Chrome)

Find Sendspark

Click the ⋮ three dots next to it

Select “Manage extension”

Right-click the Sendspark icon

Click “Manage extension”

2️⃣ Turn on file access

Scroll down

Find “Allow access to file URLs”

Toggle it ON ✅

3️⃣ Refresh and record

Go back to your PDF tab

Refresh the page

Start recording again 🎬

![Sendspark extension right-click menu highlighting the Manage Extension option](/images/14788333/3e73183f-image.png)

![Extension settings with Allow access to file URLs toggle turned on](/images/14788333/afbb1e69-image.png)

💡 Why this is needed:

Chrome blocks extensions from interacting with local files (like PDFs) for security reasons.

Enabling this setting allows Sendspark to properly display your camera bubble overlay during recordings.

⚠️ If this is not enabled

You may notice:

Your camera bubble not appearing

Your face missing from the recording
