If your camera bubble doesn't appear when recording over a PDF, it's because Chrome blocks extensions from accessing local files by default. To fix it, open the Sendspark extension's "Manage extension" page, turn on "Allow access to file URLs," then refresh your PDF tab and start recording again so your face shows over the document.

## Key takeaways

- Chrome **blocks extensions from accessing local files** (like PDFs) by default, which hides the camera bubble.
- Open the **Sendspark extension → Manage extension**.
- Turn on **"Allow access to file URLs."**
- **Refresh the PDF tab**, then start recording again.

If your camera bubble (your face) is not showing when recording over a PDF, this is due to a Chrome security setting. By default, Chrome blocks extensions from accessing files (like PDFs opened from your computer), which prevents the camera overlay from appearing.

⚙️ **Enable access to PDF files**

1️⃣ Open extension settings

Click the 🧩 puzzle icon (top right in Chrome)

Find Sendspark

Click the ⋮ three dots next to it

Select “Manage extension”

Right-click the Sendspark icon

Click “Manage extension”

2️⃣ Turn on file access

Scroll down

Find “Allow access to file URLs”

Toggle it ON ✅

3️⃣ Refresh and record

Go back to your PDF tab

Refresh the page

Start recording again 🎬
