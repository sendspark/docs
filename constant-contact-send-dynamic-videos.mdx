To send a dynamic video through Constant Contact, open your finished Dynamic Video campaign in Sendspark, click **Share Campaign**, select **Constant Contact**, and copy the code snippet. Then paste that snippet into the HTML of a Constant Contact email built from a template that allows custom code (Constant Contact's standard blocks don't include an HTML block), pick your contact list, and send.

## Key takeaways

- You share a dynamic video by copying the **Share Campaign → Constant Contact** code snippet from Sendspark and pasting it into your email's HTML.
- Constant Contact does not offer an HTML block inside its standard block-based templates, so you need an email built to accept the pasted code (a from-scratch or custom template) rather than the default drag-and-drop blocks.
- In Sendspark's Dynamic Videos tab, the bubble shows only your **original** recording (sometimes with a banner). That is expected. The banner does not appear to recipients, and the background swaps per contact, so what the recipient sees is what you set on the share page.
- Merge tags populate automatically for the recipient: first name, company name, and job title render on the landing page (header, message, and call-to-action).

## How to send dynamic videos in Constant Contact

1. **Finish your Dynamic Video campaign** in Sendspark. (New to this? See [Constant Contact — Send Personalized Videos](/constant-contact-send-personalized-videos) for building the campaign from scratch.)
2. Click **Share Campaign**, type or select **Constant Contact**, and copy the code snippet it provides.
3. In Constant Contact, open or build an email **template that supports pasting HTML code**. Constant Contact's standard templates only offer drag-and-drop blocks with no HTML block, so use a from-scratch or custom template set up to accept the snippet.
4. **Paste the code snippet** into your template's HTML where you want the video preview to appear.
5. Click **Continue**, select the **contact list** you generated videos for, and review the preview.
6. Click **Send**. When the email arrives, the animated preview links to the personalized landing page, with merge tags (first name, company, job title) filled in per recipient.

## Frequently asked questions

<AccordionGroup>
  <Accordion title="Why can't I add the video as a block in Constant Contact?">
    Constant Contact's standard templates are block-based (drag-and-drop) and do not include an HTML block, so there is no in-between option to drop the snippet in as a block. You need an email built from scratch or a custom template that lets you paste HTML code, then paste the Sendspark snippet into that code.
  </Accordion>
  <Accordion title="In Sendspark the preview shows my original video with a banner. Is that a problem?">
    No. The bubble in the Dynamic Videos tab shows only your original recording, and it may display a banner. The banner does not show to recipients and the background changes dynamically per contact. What the recipient sees is whatever you set on the share page, so review that page before sending.
  </Accordion>
  <Accordion title="Do merge tags work when sending through Constant Contact?">
    Yes. On the recipient's landing page the merge tags populate automatically: first name in the header, company name in the message, and job title in the call-to-action.
  </Accordion>
  <Accordion title="How long does the video take to arrive?">
    In the walkthrough the email arrived within a few minutes of sending. Delivery time can vary with your Constant Contact sending.
  </Accordion>
</AccordionGroup>
