If you're on a Sendspark agency plan and want to run multiple workspaces, open the menu and choose **Change Workspace**, then click **Create Workspace**. After finishing setup, copy the new Workspace ID and share it with Tech Support in the chat, tell them how many minutes to allocate to the workspace, and they'll transfer them for you.

## Key takeaways

- These steps are for agencies that plan to run multiple workspaces and have subscribed to an agency plan.
- Create a workspace via the menu → **Change Workspace** → **Create Workspace**.
- After setup, copy the **Workspace ID** and share it with Tech Support through the chat system to activate it.
- If you're on an agency plan, let Tech Support know how many minutes you'd like allocated to the workspace and they'll transfer them.

If you are an agency planning to run multiple workspaces and you've subscribed to an agency plan, follow these steps to activate a new workspace:

1- Head to your menu button and hit CHANGE WORKSPACE

![Menu](/images/11799235/7d2787d8-image.png)

2- Click on CREATE WORKSPACE

![Change Workspace](/images/11799235/afbe6b8d-image.png)

3- Once you finish the required steps, you'll get the following window. Please copy the Workspace ID and share it with Tech Support through the chat system. If you need to head back, click Switch Workspace to go back to your previous workspace:

![Copy Workspace](/images/11799235/e021b7e4-image.png)

If you currently have an agency plan, please let us know how many minutes you’d like to allocate to that workspace, and we’ll transfer them for you
