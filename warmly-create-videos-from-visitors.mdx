You can automatically send personalized videos to anonymous website visitors using Warmly and Sendspark.

Start by watching the video below, and referencing the steps for additional context.

## How to Send AI Personalized Videos to Warmly Leads

<iframe src="https://www.youtube.com/embed/_ek5EhymqUY?rel=0" allow="autoplay; fullscreen; picture-in-picture; encrypted-media" allowFullScreen />

1. Start by set up your Warmly Orchestrator for "Identified contact visited website".

![Warmly Orchestrator workflow setup for identified contact visited website trigger](/images/11382818/94ecc380-image.png)

2. Add an action to send a webhook at the end of the workflow.

![Warmly Orchestrator action panel with webhook configuration option selected](/images/11382818/c1b8c268-image.png)

3. Create an [Automated Workflow](/sendspark-automated-workflows-beginner-s-guide)in Sendspark. Use "Webhook" as a trigger, and add this webhook URL to the Warmly Orchestrator.

![Sendspark automated workflow creation interface with webhook trigger selection](/images/11382818/04e2331d-image.png)

4. Add an action to your Sendspark automation to create a [dynamic video](/dynamic-videos-make-videos-with-ai-introductions). Link to the Dynamic Video Campaign that you would like to use.

![Sendspark workflow action panel linking to dynamic video campaign](/images/11382818/3f906d90-image.png)

5. Add a follow up action to send this video out in email. Note that there are many ways you can send [videos in email](https://help.sendspark.com/en/collections/12626407-automatically-send-videos) after they are generated.

Let us know if you need any help with this setup!
