To send dynamic videos in Reply, create your Dynamic Video campaign in Sendspark, create a new template in Reply, then copy the **Reply** snippet from Sendspark and paste it into the template's code view. Each recipient receives their own personalized video automatically.

## Key takeaways

- Sendspark's Dynamic Video feature turns one recording of yourself into personalized videos placed over any background or URL, ideal for sales outreach, demos, onboarding, and customer success.
- You need two things: a **Reply template** and the **Reply snippet** Sendspark generates for the campaign.
- Paste the snippet in Reply's **code view mode** so it renders as the video preview.
- For an image-and-link approach using dynamic variables (first name, company, job title), see [Reply — Send Personalized Videos at Scale](/reply-send-personalized-videos-at-scale).

In this tutorial, we'll show you how to send dynamic videos in Reply using the Sendspark + Reply integration.

Sendspark's Dynamic Video feature lets you take one video of yourself and instantly put that video over any background, so it looks like you're recording over any website or URL.

Use this to supercharge your sales outreach, product demos, onboarding, customer success, and more!

## How to Share Dynamic Videos in Reply

1. Go to Reply and create a new Template

2. Return to Sendspark, and copy the "Reply" snippet to your clipboard

3. Click the code view mode and paste the code right there.

4. Simple as that!!
