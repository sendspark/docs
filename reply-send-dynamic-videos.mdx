In this tutorial, we'll show you how to send dynamic videos in Reply using the Sendspark + Reply integration.

Sendspark's Dynamic Video feature lets you take one video of yourself and instantly put that video over any background, so it looks like you're recording over any website or URL.

Use this to supercharge your sales outreach, product demos, onboarding, customer success, and more!

## How to Share Dynamic Videos in Reply

1. Go to Reply and create a new Template

2. Return to Sendspark, and copy the "Reply" snippet to your clipboard

3. Click the code view mode and paste the code right there.
