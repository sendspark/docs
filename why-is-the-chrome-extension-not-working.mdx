If the Sendspark Chrome Extension won't record, it's usually because you're on a page Google blocks screen recorders from: the Chrome Web Store, an empty or new tab, or the chrome://extensions page. Navigate to a normal webpage with content and try recording again — that resolves most cases.

## Key takeaways

- Google blocks screen recorders on certain pages, so recording fails there regardless of your setup.
- The three most common blocked pages are the **Chrome Web Store**, an **empty/new tab**, and your **chrome://extensions** library.
- The fix in each case is the same: navigate to a normal webpage that has content and try recording from there.
- If it still doesn't work on a normal page, reach out through the in-app chat for help.

Have no fear, there are plenty of completely different reasons why the Chrome Extension might not be working for you. Let's fix this!

## 1. You're recording over the Sendspark listing in the Web Store ❌

For some reason, Google doesn't let screen recorders work directly in the Web Store. If you try to record over the Sendspark listing - or any other Chrome Extension listing - it will not work.

Try navigating to a new tab, like this one, and see if you can record from there.

## 2. You're recording over a new or empty tab ❌

Unfortunately, the Chrome extension will not work over an empty tab in Chrome. Try navigating to a different webpage that has content, like this one, and see if it works from there.

## 3. You're recording over your Chrome Extension library ❌

Google does not allow Chrome Extensions to record over the chrome://extensions page. Try recording over this page, and see if it works.

## 4. Something else?

If you're still having trouble, please let us know asap through the chat feature here. We will do our best to help!
