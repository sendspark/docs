To create a Sendspark API key, open **API Credentials** from your account menu (or go straight to [sendspark.com/settings/api-credentials](https://sendspark.com/settings/api-credentials)) and click **GENERATE KEYS**. If you see a badge next to the feature, your plan needs an upgrade to access API keys.

## Key takeaways

- Find your keys under **API Credentials** in your account menu, or go directly to [sendspark.com/settings/api-credentials](https://sendspark.com/settings/api-credentials).
- Click **GENERATE KEYS** to create a new key.
- A badge next to a feature's title means an upgrade is required to use it.
- API keys authenticate requests to the Sendspark API, which you can use to [automatically create dynamic videos](/api-automatically-create-dynamic-videos-via-api).

API keys are easily accessible through the API section of your menu.

You can use this link for fast access https://sendspark.com/settings/api-credentials,

Or you can click on your menu and select API Credentials:

![Sendspark menu with API Credentials option highlighted](/images/11407318/f86416a4-image.png)

Once there, you can use the GENERATE KEYS button in order to create a new key:

![API Credentials page with GENERATE KEYS button](/images/11407318/74800531-image.png)

Keep in mind that if you see a badge next to a feature's title, it means an upgrade is required to use it.
