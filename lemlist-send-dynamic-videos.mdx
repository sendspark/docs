## 1. Create your dynamic video campaign in Sendspark

If you have not done that yet, please start with [this article](/dynamic-videos-make-videos-with-ai-introductions)!

## 2. Select "Share Campaign"

![Sendspark Share Campaign button highlighted in the interface](/images/8420966/9090a2ae-image.png)

## 3. Select "Lemlist"

![Sendspark integration options menu with Lemlist option visible](/images/8420966/47326a32-image.png)

## 4. Go to your email campaign in Lemlist

![Lemlist email campaign editor interface with compose area](/images/8420966/cff93c7d-image.png)

## 5. Select the "..." on the bottom of your editor to get "More Rich" options

![Lemlist editor bottom toolbar showing three dots More Rich options menu](/images/8420966/3c5423b0-image.png)

## 6. Select "Code View"

![Lemlist Code View option selected in the rich formatting menu](/images/8420966/11c0a058-image.png)

## 7. Paste the snippet from Sendspark

![Sendspark video snippet pasted into Lemlist code editor](/images/8420966/9240d698-image.png)

## 8. Switch back out of Code View

![Lemlist editor showing exit from Code View back to visual editor](/images/8420966/e9ce3aea-image.png)

## 9. You should see your dynamic video preview in email

![Lemlist email preview displaying dynamic video embedded in message](/images/8420966/24829e8a-image.png)

That's it! If you want to double check that you set things up correctly before sending the email, click "preview" in bottom right.

![Lemlist preview button located in the bottom right corner](/images/8420966/19759a7c-image.png)

When you preview this email for contacts for whom you have generated videos for, you should see their AI video fill in the preview area.

![Lemlist email preview showing personalized AI video loaded for contact](/images/8420966/cb2621f1-image.png)

If you do NOT see their video, double check that you have the **same email address for the contact in both Sendspark and Lemlist.**

We use the email address to match the right video to the right person, so if you are missing their email address in Sendspark, you would just see the fallback video in Lemlist.
