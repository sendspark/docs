To use the Teleprompter, open it from the Sendspark Chrome Extension and add the text you want to read. The moment you click **Start Recording**, the prompter begins scrolling down so you can read your script while filming. You can click into the prompter window at any time to edit the text.

## Key takeaways

- The Teleprompter opens from the Sendspark Chrome Extension and scrolls your script automatically once you click **Start Recording**.
- If you click away and lose the window, right-click the Chrome icon and select the Teleprompter from the menu to reopen it.
- You can click directly in the prompter window to edit the text and adjust it to your needs.
- Need script ideas? The **WELCOME** section offers AI script templates you can load into the Teleprompter via the **Script** button.

## How to manipulate the Teleprompter from the Chrome Extension

![Sendspark extension with red arrow pointing to the Teleprompter document icon](/images/11178503/d5f3f63c-image.png)

Clicking there will open the Teleprompter:

![Sendspark Teleprompter window showing script text and Start Recording button](/images/11178503/b66ad92d-image.png)

This window lets you add the text you want to use, and the moment you click on "Start Recording" the prompter will begin to scroll down.

If you accidentally clicked away, just make a right click on the Chrome icon, and you'll see the Teleprompter within the options to select

![Right-click context menu on Chrome icon showing Teleprompter selection option](/images/11178503/6439b799-image.png)

## Are you looking for some fresh ideas for your videos?

Head to WELCOME and click on one of the Templates, and you'll get some AI scripts to use in different scenarios:

![Welcome section displaying available video script templates](/images/11178503/e05ec6f5-image.png)

You can select between these options, and once you are ready, head to READY TO GO:

![Ready to Go button to launch Chrome extension with selected template](/images/11178503/16ae6887-image.png)

Clicking that button will fire up the Chrome Extension, and if you click the Script button, the Template you selected before will show up:

![Teleprompter window displaying loaded template script ready for editing](/images/11178503/4e300201-image.png)

You can click on the prompter window to edit the text and better adjust it to your needs.

And that's it! You are now ready to start recording :)
