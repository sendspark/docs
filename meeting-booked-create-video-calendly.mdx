---
title: "Meeting Booked - Create Video (Calendly)"
description: "Learn how to create automated AI videos when meetings are booked using Calendly triggers and Sendspark workflows integration."
---

## 1. Create Automation

Go to [Sendspark Automated Workflows](https://sendspark.com/automated-workflows).

Create a "new automation"

![New automation button in Sendspark Automated Workflows](/images/11158045/b49507ea-image.png)

## 2. Add Trigger: Meeting Booked

Add a trigger to start the automation.

![Empty trigger step prompting to add a trigger for the selected app](/images/11158045/75d38c4f-image.png)

Select **Calendly**, and the event **"Invitee Created."**This event will fire anytime someone books a meeting (and gets created as an invitee)

![Trigger setup with Calendly app and Invitee created event selected](/images/11158045/2c4c5161-image.png)

## 3. Add Action: Create Sendspark AI Video

Add an action. Select **"Sendspark"**and **"Create Dynamic Video"**

![Action setup with Sendspark app and Create dynamic video action selected](/images/11158045/b2baca6c-image.png)

Map the fields from Calendly into Sendspark in order to use in the AI video personalization.

![Calendly invitee Name and Email fields mapped into the video campaign](/images/11158045/028e4bb7-image.png)

## 4. Add Action: Send Video

You can send this video directly in Gmail or Outreach, or check out one of the [many other ways](https://help.sendspark.com/en/collections/12626407-automatically-send-videos) to share your video in automated workflows.

![Gmail send email action embedding the dynamic video into the message body](/images/11158045/9fbd6d00-image.png)
