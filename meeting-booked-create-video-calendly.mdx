When someone books a meeting through Calendly, Sendspark can automatically create a personalized AI video and send it. You build this in Sendspark's Automated Workflows: add a Calendly **"Invitee Created"** trigger, add a **"Create Dynamic Video"** action that maps Calendly fields for personalization, then send the video in Gmail, Outreach, or another connected channel.

## Key takeaways

- A workflow pairs one **trigger** (Calendly's **Invitee Created** event) with actions that create and send the video.
- The **Invitee Created** event fires anytime someone books a meeting and is created as an invitee.
- Map the fields from Calendly into Sendspark so the AI video is personalized to the person who booked.
- Send the finished video directly in Gmail or Outreach, or use one of the other automated ways to share videos.

## 1. Create Automation

Go to [Sendspark Automated Workflows](https://sendspark.com/automated-workflows).

Create a "new automation"

![New automation button in Sendspark Automated Workflows](/images/11158045/b49507ea-image.png)

## 2. Add Trigger: Meeting Booked

Add a trigger to start the automation.

![Empty trigger step prompting to add a trigger for the selected app](/images/11158045/75d38c4f-image.png)

Select **Calendly**, and the event **"Invitee Created."**This event will fire anytime someone books a meeting (and gets created as an invitee)

![Trigger setup with Calendly app and Invitee created event selected](/images/11158045/2c4c5161-image.png)

## 3. Add Action: Create Sendspark AI Video

Add an action. Select **"Sendspark"**and **"Create Dynamic Video"**

![Action setup with Sendspark app and Create dynamic video action selected](/images/11158045/b2baca6c-image.png)

Map the fields from Calendly into Sendspark in order to use in the AI video personalization.

![Calendly invitee Name and Email fields mapped into the video campaign](/images/11158045/028e4bb7-image.png)

## 4. Add Action: Send Video

You can send this video directly in Gmail or Outreach, or check out one of the [many other ways](https://help.sendspark.com/en/collections/12626407-automatically-send-videos) to share your video in automated workflows.

![Gmail send email action embedding the dynamic video into the message body](/images/11158045/9fbd6d00-image.png)

## Frequently asked questions

<AccordionGroup>
  <Accordion title="Which Calendly event triggers the video?">
    Use the **Invitee Created** event. It fires anytime someone books a meeting and is created as an invitee in Calendly, which is what starts the automation.
  </Accordion>
  <Accordion title="Do I need to record a video first?">
    No. The workflow's **Create Dynamic Video** action generates the Sendspark video automatically when the trigger fires, using the fields you map in from Calendly.
  </Accordion>
  <Accordion title="How is the video personalized?">
    In the Create Dynamic Video action you map fields from Calendly into Sendspark. Those fields are then used in the AI video personalization for each person who books.
  </Accordion>
  <Accordion title="How do I send the video after it's created?">
    You can send it directly in Gmail or Outreach as an action, or use one of the [many other ways](https://help.sendspark.com/en/collections/12626407-automatically-send-videos) to share videos in automated workflows.
  </Accordion>
  <Accordion title="Where do I build this automation?">
    In the [Sendspark Automated Workflows](https://sendspark.com/automated-workflows) section. Click **New Automation** to start from scratch.
  </Accordion>
</AccordionGroup>
