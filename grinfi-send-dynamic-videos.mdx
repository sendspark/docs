Sendspark makes it easy to send AI-personalized videos in Grinfi.

So you can create just one placeholder video in Sendspark, and when you send it in Grinfi, every recipient will receive a video with their own name and website in it.

## How to Send AI-Personalized Videos in Grinfi

## 1. Create a Dynamic Video Campaign in Sendspark

Go to the [dynamic video section](https://sendspark.com/dynamic-videos) of Sendspark.

Click **"Create Dynamic Video"** to create a new campaign: **[https://sendspark.com/dynamic-videos](https://sendspark.com/dynamic-videos)**

![Sendspark dynamic video creation page with Create Dynamic Video button](/images/12534232/2490e6fe-image.png)

From here, you can add a video and select options to...

- Use AI to clone your voice, so you can automatically greet each viewer by name
- Simulate scrolling through different website pages
- and more!

![Dynamic video editor interface showing AI voice cloning and website scrolling options](/images/12534232/349ca19c-image.png)

You can also customize video landing pages, so prospects will see more than just the video, and be able to book meetings or take actions immediately after watching.

![Custom video landing page editor with booking and call-to-action features](/images/12534232/87fed690-image.png)

Learn more about [creating dynamic videos here](/dynamic-videos-make-videos-with-ai-introductions).

## 2. Import Contacts

You can create dynamic videos for contacts in a few ways:

1. Manually type contact data
2. Import a CSV
3. Generate videos via automation (Zapier, API, etc)

Once dynamic videos have been created, you'll see them populate, and you can play them to test out the AI and make sure everything looks great.

![Dynamic video campaign list showing generated personalized videos for contacts](/images/12534232/68c997d9-image.png)

## 3. Share in Grinfi

To share videos in Grinfi, simply click **"Share Campaign."**

Then, select **"**Grinfi**" to copy the snippet code**

![Share Campaign dialog with Grinfi selected and snippet code ready to copy](/images/12534232/524535b0-image.png)

Now, you can take that snippet and paste it into an email in Grinfi by using the HTML mode:

![Grinfi email editor in HTML mode with video snippet code pasted](/images/12534232/5491efe8-paste.gif)

Feel free to adjust the size with this part of the snippet:

![HTML snippet code showing width and height parameters for video sizing](/images/12534232/3ec45928-50.gif)

Notice that what you see is simply a placeholder that will change according to every video you create with an email address inside of the dynamic video campaign in Sendspark:

![Dynamic video placeholder example showing personalization with recipient name and website](/images/12534232/22978cbe-image.png)
