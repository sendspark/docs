Can I use an image instead of a video in a Sendspark dynamic video?

![Google Drive share dialog with Anyone with the link can view option selected](/images/9552587/a44b87b4-image.png)

Absolutely! While Sendspark is built for creating dynamic videos, you can still leverage its personalization features with images.

<iframe src="https://www.youtube.com/embed/w45xyDoeodE?rel=0" allow="autoplay; fullscreen; picture-in-picture; encrypted-media" allowFullScreen />

## Here's how to use an image as a background in your Sendspark dynamic video:

**1. Prepare your image: Upload your desired image to Google Drive.**  
​

**2. Share the image publicly:**

In Google Drive, right-click on the image and select "Share."

Under the "Get link" section, change access to "Anyone with the link can view."

Click "Copy link" to obtain the public sharing URL.

![Google Drive file link URL format showing original and edited link examples](/images/9552587/a31ee1a0-image.png)

**3. Create your Sendspark dynamic video:**

Follow the usual process for creating a Dynamic Video Campaign in Sendspark.

This will only work if this is an image file (PNG, JPG):

Basically you remove the "file/d/xxxxx/view etc..." and you add

"uc?export=view&id=" followed by the same ID

**4. Additional Tips:**

-Consider using a visually appealing and high-resolution image for best results.  
​

-Remember to keep the public sharing link active on Google Drive while the campaign is in process.
