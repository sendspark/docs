Yes, you can use an image instead of a video as your dynamic video background in Sendspark. Upload the image to Google Drive, set sharing to "Anyone with the link can view," then edit the share link into a direct format and paste it into the Background URL field. Personalization features still apply.

## Key takeaways

- Sendspark is built for dynamic videos, but you can still use its personalization features with a static image as the background.
- The image must be publicly accessible: in Google Drive, set link access to "Anyone with the link can view."
- For image files (PNG, JPG), rewrite the share link to the direct format `https://drive.google.com/uc?export=view&id=FILE_ID`.
- For PDFs or other non-image files, change the end of the link from `/view` to `/preview` (you can't avoid seeing the zoom controls in this case).
- Keep the public sharing link active for the whole time the campaign is running.

## Here's how to use an image as a background in your Sendspark dynamic video:

**1. Prepare your image: Upload your desired image to Google Drive.**  
​

**2. Share the image publicly:**

In Google Drive, right-click on the image and select "Share."

Under the "Get link" section, change access to "Anyone with the link can view."

Click "Copy link" to obtain the public sharing URL.

![Google Drive file link URL format showing original and edited link examples](/images/9552587/a31ee1a0-image.png)

**3. Create your Sendspark dynamic video:**

Follow the usual process for creating a Dynamic Video Campaign in Sendspark.

## Frequently asked questions

<AccordionGroup>
  <Accordion title="Can I use an image instead of a video in a dynamic video campaign?">
    Yes. While Sendspark is built for dynamic videos, you can still use its personalization features with an image. Set the image as your Background URL and personalize as you normally would.
  </Accordion>
  <Accordion title="Why does my image need to be publicly shared?">
    Sendspark pulls the image from the URL you provide, so it has to be reachable without a login. In Google Drive, right-click the image, choose "Share," and set access to "Anyone with the link can view."
  </Accordion>
  <Accordion title="How do I format a Google Drive link for an image (PNG, JPG)?">
    Remove the `file/d/xxxxx/view...` portion of the share link and replace it with `uc?export=view&id=` followed by the same file ID. For example, `https://drive.google.com/file/d/FILE_ID/view?usp=sharing` becomes `https://drive.google.com/uc?export=view&id=FILE_ID`.
  </Accordion>
  <Accordion title="What if my file is a PDF or another non-image file?">
    Use the same share link but change the ending from `/view` to `/preview`. Note that with non-image files you can't avoid seeing the zoom controls in the background.
  </Accordion>
  <Accordion title="Do I need to keep the Google Drive link active during the campaign?">
    Yes. Keep the public sharing link active on Google Drive for as long as the campaign is running, otherwise the background image won't load. Using a high-resolution image also gives the best results.
  </Accordion>
</AccordionGroup>
