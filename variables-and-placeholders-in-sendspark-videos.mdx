Sendspark supports two kinds of variables for personalizing videos: text variables and voice (AI voice cloning) variables. Text variables — first name, company name, email, and a custom field — personalize the video landing page template. AI voice cloning currently supports only the first name variable, used once at the very beginning of the dynamic video.

## Key takeaways

- Sendspark has two variable types: **text variables** and **AI voice-cloning (voice) variables**.
- Text variables available for landing page templates are first name, company name, email, and a custom field.
- AI voice cloning currently supports only the first name variable, and only once at the beginning of the video.
- You can only replace the specified variables — you can't dynamically replace whole sentences.

**Using Dynamic Variables in Sendspark**

**Two Types of Variables: Text and Voice Variables (Or Placeholders)**

**1. Text Variables:** These are used in the templates for the landing page of videos that Sendspark generates. The available options include First name, Company name, email, and a Custom Field

![](/images/11048080/79e76d2e-image.png)

**2. AI Voice Cloning:** Currently, this supports only the first name variable. It can be used once and only at the beginning of the dynamic video.

![](/images/11048080/5b761a52-image.png)

While Sendspark is exploring adding more variables in the future, it's important to note that you can't dynamically replace whole sentences, only the specified variables.
