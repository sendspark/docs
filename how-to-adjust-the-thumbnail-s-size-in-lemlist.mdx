## How to Adjust a Thumbnail in Lemlist

Step by step guide on how to adjuste a thumbnail in Lemlist to fit perfectly into your email campaigns.

1. **Accessing the Email Editor:**
  - Log into your Lemlist account and navigate to the email editor where you want to adjust the thumbnail.
2. **Inserting the Thumbnail:**
  - Paste the thumbnail code into your email body. By default, it might appear too large for the email layout.
3. **Switching to Code View:**
  - If you don't see the code view toolbar, click on the three dots menu and select "Code View."
4. **Editing the Thumbnail Size:**
  - Locate the `<h>` reference in the code where your thumbnail is inserted. Adjust the `width` attribute to resize the thumbnail accordingly.
    - For example, setting `width="50%"` will make it half the original size, while `width="30%"` will reduce it further.
5. **Previewing and Finalizing:**
  - Switch back to the code view to see how the thumbnail looks after resizing. Use the preview option to view the email layout with the adjusted thumbnail.

## Tips for Optimal Thumbnail Size:

- **Proportionality:** Ensure the thumbnail size is appropriate for the email format to maintain visual balance.
- **Testing:** Experiment with different width percentages (e.g., 50% or 30%) to find the ideal size that suits your campaign's aesthetic.

Adjusting thumbnails in Lemlist is a straightforward process that enhances the visual appeal of your email campaigns.

By resizing thumbnails to fit the email layout, you can create more engaging and professional-looking communications.
