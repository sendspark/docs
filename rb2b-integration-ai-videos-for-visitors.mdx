The RB2B and Sendspark integration automatically creates AI-personalized videos for the website visitors RB2B identifies. You build a Sendspark Agentic Workflow triggered by an incoming webhook, paste that webhook URL into RB2B, map the visitor data into a Create Dynamic Video step, then add a step to send the video by email or LinkedIn.

## Key takeaways

- The integration turns website visitors identified by RB2B into personalized videos automatically, on autopilot.
- It's built as a Sendspark Agentic Workflow using an **Incoming Webhook** trigger.
- You copy the Sendspark webhook URL into RB2B's integrations, then send a test event to confirm the connection and pull RB2B data into the workflow.
- Map the RB2B data fields into a **Sendspark: Create Dynamic Video** step, then add a final step to send the video through email, LinkedIn, or any of 50+ sending platforms.

Use the RB2B and Sendspark integration to send personalized videos to website visitors on autopilot!

## How to Use the RB2B + Sendspark Video

1. Go to Sendspark [Agentic Workflows](https://sendspark.com/agentic-workflows) and create a new workflow
2. For the trigger, select**Incoming Webhook**
3. Copy the webhook URL to your clipboard, and click **"wait for webhook"** to start testing your webhook ​![Sendspark Agentic Workflows interface showing webhook URL field and wait for webhook button](/images/12064378/322c2f97-image.png)
4. Go to [RB2B integrations](https://app.rb2b.com/integrations)
5. Select the **Sendspark**, and paste the URL from Sendspark into RB2B ​![RB2B integrations page with Sendspark selected and webhook URL paste field](/images/12064378/2db24444-image.png)
6. Click **Send test event to your Sendspark account** to test the webhook connection![RB2B Sendspark integration settings with Send test event button highlighted](/images/12064378/c4e6be64-image.png)
7. Back in Sendspark, you should see data from RB2B. You can now use that data to create an AI personalized video!
8. Add another step in your automation for **Sendspark: Create Dynamic Video**
9. Select the [dynamic video campaign](/dynamic-videos-make-videos-with-ai-introductions) from Sendspark you would like to use, and map in the data fields from RB2B. These are the data fields that you want use to personalize your Sendspark video. ​![Sendspark Create Dynamic Video step showing data field mapping from RB2B](/images/12064378/d3a87968-image.png)
10. Add another step in your automation to send the video out through email or LinkedIn. Sendspark integrates with 50+ sending platforms, so you can choose your platform, or just send with Gmail/Outlook. [Learn more here](/sendspark-automated-workflows-beginner-s-guide).

Happy scaling!
