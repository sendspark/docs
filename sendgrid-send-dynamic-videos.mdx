Sendspark lets you send AI-personalized videos in SendGrid from a single placeholder video. Create a dynamic video campaign in Sendspark, import your contacts, then click Share Campaign and select SendGrid to get a code snippet. In SendGrid's Design Editor, drop in a CODE module, paste the snippet, and each recipient gets their own personalized video.

## Key takeaways

- One placeholder video in Sendspark becomes a personalized video for every recipient, each with their own name and website.
- Create the dynamic video campaign in Sendspark, then import contacts by manual entry, CSV upload, or automation (Zapier, API, etc).
- Get the SendGrid snippet from **Share Campaign → SendGrid**.
- In SendGrid, use the **Design Editor**, add a **CODE** module under BUILD, paste the snippet, and click Update — the placeholder automatically turns into the right video.

Sendspark makes it easy to send AI-personalized videos in SendGrid.

So you can create just one placeholder video in Sendspark, and when you send it in SendGrid, every recipient will receive a video with their own name and website in it.

## How to Send AI-Personalized Videos in SendGrid

## 1. Create a Dynamic Video Campaign in Sendspark

Go to the [dynamic video section](https://sendspark.com/dynamic-videos) of Sendspark.

Click **"Create Dynamic Video"** to create a new campaign: **[https://sendspark.com/dynamic-videos](https://sendspark.com/dynamic-videos)**

![Sendspark dynamic video section with Create Dynamic Video button](/images/12511171/b91e3ce6-image.png)

From here, you can add a video and select options to...

- Use AI to clone your voice, so you can automatically greet each viewer by name
- Simulate scrolling through different website pages
- and more!

![Dynamic video editing interface with AI voice cloning and website scrolling options](/images/12511171/b46848da-image.png)

You can also customize video landing pages, so prospects will see more than just the video, and be able to book meetings or take actions immediately after watching.

![Custom video landing page builder with booking and action elements](/images/12511171/c55a3e63-image.png)

Learn more about [creating dynamic videos here](/dynamic-videos-make-videos-with-ai-introductions).

## 2. Import Contacts

You can create dynamic videos for contacts in a few ways:

1. Manually type contact data
2. Import a CSV
3. Generate videos via automation (Zapier, API, etc)

![Contact import options showing manual entry, CSV upload, and automation methods](/images/12511171/088cc31a-image.png)

Once dynamic videos have been created, you'll see them populate, and you can play them to test out the AI and make sure everything looks great.

![List of generated dynamic videos with play buttons for testing personalization](/images/12511171/179535e0-image.png)

## 3. Share in Sendgrid

To share videos in SendGrid, simply click **"Share Campaign"**

Then, select **"**Sendgrid**"**

![Share Campaign menu with SendGrid selected as distribution option](/images/12511171/eafa0e0d-image.png)

Now, you can take that snippet and paste it into an email in SendGrid. Select the DESIGN EDITOR for a better experience:

![SendGrid email editor with Design Editor option highlighted](/images/12511171/7a576654-image.png)

Once in the email builder section, click on BUILD and grab the CODE module and drop it in the email

![SendGrid email builder showing BUILD menu and CODE module selection](/images/12511171/cf08b8ed-image.png)

Now simply paste the code you copied from the share dynamic video campaign button and click on UPDATE

![SendGrid code editor with Sendspark video code pasted and Update button](/images/12511171/0c079161-image.png)

Now click back on design. And you'll get something like this, a placeholder that will automatically turn into the right video.

![SendGrid email preview showing dynamic video placeholder ready for campaign launch](/images/12511171/80ed705c-image.png)

And that's all you need! You can now launch your campaign
