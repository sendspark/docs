To create Dynamic Background videos without an AI voice greeting, turn off name personalization before you select your video. Click **Create Dynamic Video**, and when prompted to choose the video, switch the name personalization option OFF first.

## Key takeaways

- Turn off name personalization when you want dynamic background videos in bulk but without an AI-spoken name greeting.
- "Name personalization" and "voice cloning" are the same setting, and it is ON by default.
- Toggle it OFF before selecting your video in the Create Dynamic Video flow.

THis help article shows you how to turn the Voice Cloning feature off, in case you want to create videos with Dynamic Backgrounds in bulk but with no Name Personalization as a greeting.

Simply hit the CREATE DYNAMIC VIDEO button and when prompted to select the video you want to use, BEFORE doing it, make sure you turn OFF the name personalization option:

## Frequently asked questions

<AccordionGroup>
  <Accordion title="Is name personalization the same as voice cloning?">
    Yes. In the Dynamic Videos flow, "name personalization" is the same thing as voice cloning. See [Toggle Voice Cloning On/Off](/toggle-voice-cloning-on-off).
  </Accordion>
  <Accordion title="Is voice cloning on by default?">
    Yes, name personalization (voice cloning) is ON by default, so you only need to turn it off if you don't want an AI-spoken name greeting.
  </Accordion>
  <Accordion title="What happens when I turn it off?">
    Your videos still generate with a dynamic background for each viewer, but without the AI voice greeting that says the recipient's name.
  </Accordion>
</AccordionGroup>
