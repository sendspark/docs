---
title: "How to turn Voice Cloning's Personalization Off"
description: "Disable Voice Cloning personalization to create bulk Dynamic Background videos without name greetings. Quick setup guide included."
---

THis help article shows you how to turn the Voice Cloning feature off, in case you want to create videos with Dynamic Backgrounds in bulk but with no Name Personalization as a greeting.

Simply hit the CREATE DYNAMIC VIDEO button and when prompted to select the video you want to use, BEFORE doing it, make sure you turn OFF the name personalization option:
