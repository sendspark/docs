You can automatically create a personalized Sendspark video whenever someone submits a form or books a meeting in HighLevel (GHL). Connect the two with your HighLevel API key, build a Sendspark automation that catches a webhook from a HighLevel workflow, then add a Create a Dynamic Video action and map the contact fields. This example uses the Form Submission trigger.

## Key takeaways

- Connect HighLevel to Sendspark using your HighLevel sub-account API key (Settings → create API below Business Profile Settings).
- The flow relies on a webhook: Sendspark provides a WebHook Catcher URL that you add as a POST webhook step in your HighLevel workflow.
- In HighLevel, this example uses Form Submission as the trigger, plus Create Contact and Webhook steps in the workflow.
- In Sendspark's Action step, choose Create a Dynamic Video, paste your Campaign ID (found in the campaign URL), turn off Light View, and map each field.

In this article, we'll help you get started with your Sendspark-Highlevel workflow automation. As an example, we'll use the Form Submission method to create a workflow  
  
First, we'll retrieve your API key from HighLevel to connect your account to Sendspark:

Select your Sub-Account and click on Settings:

![HighLevel dashboard with arrows to Sub-account selection and Settings](/images/10840573/2c54a6ed-image.png)

Create your API with the blue button right below the BUSINESS PROFILE SETTINGS and copy the API key

![Business Profile Settings with arrows to Create API key and Update Information](/images/10840573/3cce6f48-image.png)

Now, let's head to Sendspark Automated-Workflows section and click on CREATE A NEW AUTOMATION: [https://sendspark.com/automated-workflows](https://sendspark.com/automated-workflows)  
  
In the TRIGGER step, we'll make these selections and then we'll hit continue

![Set up a trigger modal with HighLevel app and Event created selected](/images/10840573/9c3463e6-image.png)

If you are connecting HighLevel for the first time, you'll be asked to provide that API key we just copied before:

![HighLevel connection editor with Location API Key field for pasting the key](/images/10840573/aa592eb1-image.png)

The system will now provide a WebHook Catcher. Please copy the URL

![Webhook catcher modal with arrow to the copy URL button](/images/10840573/63b4bcb9-image.png)

Back in HighLevel, Click on SITES -- &gt; Forms --&gt; Form Builder:

![HighLevel Sites navigation with numbered arrows to Forms then Builder](/images/10840573/646535da-image.png)

Add a new Form:

![HighLevel Forms page with arrow to the Add Form button](/images/10840573/5ad4d7a5-image.png)

By dragging and dropping the options given on the left panel, we'll be able to create a simple form:

![Form builder with Form Element panel and First Name Email Website preview](/images/10840573/8e534f4e-image.png)

After hitting the SAVE button on the top right-hand side corner, go back, head to AUTOMATION, and click on CREATE WORKFLOW:

![HighLevel Workflow List with arrows to Automation and Create Workflow](/images/10840573/a63568e3-image.png)

Double-click on ADD NEW TRIGGER to open the list of options to the right:

![Workflow builder canvas with Add New Trigger and Workflow Trigger list](/images/10840573/462e4d43-image.png)

There are a number of different ways to go, but we'll follow the FORM SUBMISSION in this example. Please click this option

![Workflow trigger search for form with arrow to Form Submitted event](/images/10840573/254152d3-image.png)

Then click on ADD FILTER:

![Form Submitted trigger settings with arrow to the Add filters link](/images/10840573/21bc2741-image.png)

Select the name of the form you just created:

![Trigger filter set to Form is Testing for the created form](/images/10840573/2baf0c5c-image.png)

We'll complete the workflow with these steps: CREATE CONTACT and WEBHOOK.

The Create Contact step will have the same fields we used in the form so we can map it and retain the same details:

![Create Contact action mapping First Name Email and Website fields](/images/10840573/19eaaae5-image.png)

The WEBHOOK step will need the POST method and the URL we copied from Sendspark

![Webhook action panel set to POST method with the Sendspark URL](/images/10840573/cce064a1-image.png)

Now back to Sendspark: We'll need set up the ACTION step by clicking it:

![Sendspark automation with arrow to Add an action step row](/images/10840573/f3bb158c-image.png)

Select Sendspark - Create a Dynamic Video - and Your Account and hit continue:

![Add an action modal with Sendspark and Create dynamic video selected](/images/10840573/0d3c0004-image.png)

You'll now need to create a new Dynamic Video Campaign or simply use one you had before, and copy the Campaign ID. Something you can extract from its URL:

![Sendspark dynamic video page with arrow to campaign ID in the URL](/images/10840573/64f12fa7-image.png)

Only this section:

![Close-up of browser URL with the dynamic video campaign ID underlined](/images/10840573/e759d104-image.png)

Paste the ID in the Action Step window:

![Map your fields modal pasting the campaign ID into Dynamic Video Campaign field](/images/10840573/8260f810-image.png)

Turn OFF the Light View and map each field correctly by selecting the correct term for the correct field:  Contact NAME --&gt; FIRST SAME and so on

![Map your fields with arrows to turn off Light view and expand the value dropdown](/images/10840573/97e76428-image.png)

Click on SAVE and your Workflow will be ready to test!

## Frequently asked questions

<AccordionGroup>
  <Accordion title="Where do I find my HighLevel API key?">
    In HighLevel, select your Sub-Account and click Settings. Create your API with the blue button right below the Business Profile Settings, then copy the API key. You'll paste it into Sendspark the first time you connect HighLevel.
  </Accordion>
  <Accordion title="What trigger does this example use?">
    This example uses the Form Submission trigger in HighLevel. There are a number of different ways to trigger the workflow, but Form Submission is used here — you add a filter and select the form you created.
  </Accordion>
  <Accordion title="How does HighLevel send the data to Sendspark?">
    Through a webhook. Sendspark provides a WebHook Catcher URL that you copy, then add as a Webhook step in your HighLevel workflow using the POST method and that URL. The HighLevel workflow completes with Create Contact and Webhook steps.
  </Accordion>
  <Accordion title="How do I connect the video campaign in the Sendspark Action step?">
    In Sendspark's Action step, select Create a Dynamic Video and your account. Create a new Dynamic Video Campaign (or use an existing one) and copy the Campaign ID from its URL. Paste the ID into the Action Step window, turn off Light View, and map each field to the correct term.
  </Accordion>
</AccordionGroup>
