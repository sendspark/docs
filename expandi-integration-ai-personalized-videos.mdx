## How to Send Sendspark Videos in Expandi LinkedIn Automation

## Prepare: Connect Sendspark and Expandi

If this is your first time using Sendspark and Expandi together, please [install the Sendspark integration in Expandi](/connect-expandi-with-sendspark-for-the-first-time).

Once you have that setup, you can follow the steps below!

## 1. Create dynamic video campaign in Sendspark

Head over to the [dynamic video section](https://sendspark.com/dynamic-videos) in Sendspark, and click to**"Create Dynamic Video."**

Give you campaign a name, and go through the steps to create a dynamic video campaign, all up to the last step where you add contacts (you will do this part in Expandi).

![Create Campaign dialog on Sendspark Dynamic Videos page](/images/10077385/e4c20da4-image.png)

If you don't know how to create a dynamic video campaign, please start with this article on [creating AI personalized videos in Sendspark](/dynamic-videos-make-videos-with-ai-introductions).

## 2. Create LinkedIn automation campaign in Expandi

Log into your Expandi account, go to the Campaigns section, and select "Add New Campaign."

![Expandi campaign type selection modal with connector and messenger options](/images/10077385/fb9a6f9c-image.png)

This integration will work with any type of Expandi campaign.

## 3. Select Sendspark Campaign in Expandi

Right below where you edit your message in Expandi, you will see the option to select a Sendspark video.

![Expandi message body with Select Sendspark video dropdown below](/images/10077385/c61a30db-image.png)

Click on this to see all of your Sendspark dynamic video campaigns. Then, select the Sendspark Dynamic Video Campaign that you want to use for this campaign.

![Sendspark dynamic video campaigns listed as thumbnails to select](/images/10077385/1170ae43-image.png)

## 4. Position the &#123;sendspark_video&#125; placeholder correctly

When you select you Sendspark video, you'll see a &#123;sendspark_video&#125; placeholder variable get automatically added to your message.

![Sendspark_video placeholder added to Expandi message body text](/images/10077385/96a0b81e-image.png)

Feel free to move this around so the Sendspark video link will be exactly where you want this when you send it out.

![Expandi Set campaign steps screen with Save messages button](/images/10077385/74af4b3a-image.png)

Make sure to hit **"Save messages"** when you're done editing! :)

## 5. Enroll contacts and turn on campaign!

Now, when you enroll contacts to this campaign, each one will automatically get an AI personalized video created for them. It will look like this when it sends out:

![Personalized Sendspark video in a LinkedIn message with positive reply](/images/10077385/7243f768-image.png)

Happy connecting! :)
