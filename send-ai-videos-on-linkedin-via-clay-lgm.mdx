This guide will walk through how you can...

1. Find leads in Clay
2. Automatically Generate an AI-personalized video with Sendspark
3. Send in a LinkedIn message or Connection Request via LaGrowthMachine

## How to Send AI-Personalized Videos in LinkedIn Automation

<iframe src="https://www.youtube.com/embed/G4SPgTVrscM?rel=0" allow="autoplay; fullscreen; picture-in-picture; encrypted-media" allowFullScreen />

1. First, you'll want to identify your leads in Clay. In this tutorial, we're using the webhook from RB2B to add people who visit our website to a Clay Table. You can use **any leads in Clay**for this automation though!
2. Then, you'll use the [Sendspark + Clay integration](/clay-integration-create-ai-videos) to **generate AI-personalized videos**. If you don't know how to make AI-personalized videos in Sendspark, start with this guide on [how to make dynamic videos](/dynamic-videos-make-videos-with-ai-introductions).
3. Once your first video generates in Clay, you'll see the status updated to **"✅ Success."** Click on this, and make a new column from the value **"Video Link"** ​![Clay table displaying video generation status with success checkmark and Video Link column](/images/9658103/1dae4846-image.png)
4. You'll use the "Video Link" in your LinkedIn message or Connection Request to send a personalized video to your prospect in the next step.
5. Use the Clay Enrichment for LaGrowthMachine to add **this prospect to an audience in LaGrowthMachine**. Make sure to map in the field **"Video Link" to the LaGrowthMachine "Custom Attribute 1" ​**![Clay enrichment mapping interface showing Video Link field mapped to LaGrowthMachine Custom Attribute 1](/images/9658103/6b81a0a4-image.png)
6. In your LaGrowthMachine workflow, **use "Custom Attribute 1"**where you want to send the Sendspark personalized video. ​![LaGrowthMachine workflow editor with Custom Attribute 1 placeholder in message template](/images/9658445/c412ee03-image.png)
7. To finalize the automation, make sure that when leads get added to the audience you're using in Clay, they will get **automatically enrolled in your LaGrowthMachine workflow.**

That's it! Let us know if you have questions!
