To create automated workflows in HubSpot, a user needs the CRM Objects and Automation permissions turned on. Go to Settings → Users & Teams (under Account Management), select the user, click Edit Permissions, then find CRM Objects and Automation in the left-hand menu and switch both ON.

## Key takeaways

- Open **Settings** from the top right-hand corner of your HubSpot Home Page.
- Under **Account Management**, go to **Users & Teams** and select the right user to enable the **Edit Permissions** button.
- In the left-hand permissions menu, turn **CRM Objects** and **Automation** ON.
- Both permissions are required to successfully create automated workflows.

## Which permissions are needed to create workflows?

You need both the **CRM Objects** and **Automation** permissions turned on for the user. Both are found in the left-hand menu after clicking Edit Permissions, and both are important when creating automated workflows.
