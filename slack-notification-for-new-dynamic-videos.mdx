To get a Slack alert each time a new Dynamic Video is created, connect Sendspark to Slack through Zapier. Create a webhook in Sendspark, catch it with Zapier's Catch Hook trigger, paste the generated URL back into your Sendspark webhook, attach it to your Dynamic Video campaign, then add a Slack "Send a Message" action.

## Key takeaways

- This setup uses a Sendspark webhook paired with a Zapier Zap (Webhooks by Zapier → Slack).
- In Sendspark, create and name a new webhook under **Menu → Webhooks**.
- In Zapier, use the **Catch Hook** trigger, copy the generated URL, and paste it back into your Sendspark webhook.
- Attach the webhook to your Dynamic Video campaign, then add Slack's **Send a Message** action and choose your channel.

**In SendSpark:**

- Go to **Menu**.
- Select **Webhooks**.
- Create a new webhook.
- Name your webhook. ![SendSpark webhook creation form with name field highlighted](/images/9483231/76966ae6-image.png)

**In Zapier:**

- Create a new Zap.
- Click on **Trigger**.
- Search for **Webhooks by Zapier**.
- Select **Catch Hook**.
- Continue and copy the generated URL![Zapier Catch Hook trigger setup page with generated webhook URL](/images/9483231/dcc1197d-image.png)

**Back in SendSpark:**

- Paste the URL in the webhook you created.
- Create the webhook ![SendSpark webhook configuration panel with URL paste field](/images/9483231/0f99154f-image.png)

**Create a Dynamic Video campaign:**

- Create a new Dynamic Video Campaign
- Go to the **Webhooks** section.
- Select the campaign you just created. ![SendSpark Dynamic Video Campaign webhooks section with campaign selection](/images/9483231/bed17272-image.png)

**In Zapier:**

- Click on **Test Trigger**. ![Zapier Test Trigger button in webhook configuration interface](/images/9483231/5ca14f59-image.png)
- Search for **Slack**.
- Select **Send a Message**. ![Zapier action selection showing Slack Send a Message option](/images/9483231/9df5ec51-image.png)
- Continue and choose your Slack channel. ![Zapier Slack channel selection dropdown menu for message delivery](/images/9483231/424e2aab-image.png)
- Customize the message.
- Click on **Continue** and confirm the notification in Slack. ![Zapier notification confirmation page with Continue button](/images/9483231/b3ed2e0e-image.png)
- **DONE!!**

## Frequently asked questions

<AccordionGroup>
  <Accordion title="How do I create the webhook in Sendspark?">
    Go to **Menu**, select **Webhooks**, create a new webhook, and give it a name. You'll paste the Zapier URL into this webhook in a later step.
  </Accordion>
  <Accordion title="Which Zapier trigger connects to the Sendspark webhook?">
    In Zapier, create a new Zap, click **Trigger**, search for **Webhooks by Zapier**, and select **Catch Hook**. Continue, then copy the generated URL and paste it into the webhook you created in Sendspark.
  </Accordion>
  <Accordion title="How does the webhook know which videos to report?">
    After creating a Dynamic Video campaign, go to the **Webhooks** section and select that campaign. This attaches your webhook so it fires for new videos in that campaign.
  </Accordion>
  <Accordion title="How do I send the notification to Slack?">
    In Zapier, click **Test Trigger**, then search for **Slack** and select **Send a Message**. Continue, choose your Slack channel, customize the message, and confirm the notification in Slack.
  </Accordion>
</AccordionGroup>
