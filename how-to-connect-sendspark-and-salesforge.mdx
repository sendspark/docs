To connect Sendspark and Salesforge, open your settings menu, click **Integrations**, scroll to Sendspark, and click **Connect**. You'll be prompted for your API key, secret key, and Workspace ID — all found in Sendspark under **Settings → API Credentials**. Once connected, videos generate via API automatically for your campaigns.

## Key takeaways

- Connecting the two accounts lets you generate videos via API automatically for your campaigns.
- You connect from the **Integrations** section of your settings menu — find Sendspark and click **Connect**.
- Connecting requires three values: your **API key**, **secret key**, and **Workspace ID**.
- All three are created and found in Sendspark at [Settings → API Credentials](https://sendspark.com/settings/api-credentials).

Check this article if you have already connected your account and you want to [launch your first campaign](/salesforge-auto-generate-dynamic-videos)

## By connecting Sendspark and Salesforce, you'll be able to generate videos via API automatically

Head to your settings menu in Salesforce, then click Integrations:

![Salesforce settings menu with Integrations option highlighted](/images/12930542/4bb854e4-image.png)

Scroll down until you find Sendspark and click "Connect".

![Sendspark integration in Salesforce with Connect button visible](/images/12930542/fe47d923-image.png)

You'll be prompted to enter your API key, secret key, and the Workspace ID.

![Sendspark connection form requesting API key, secret key, and Workspace ID](/images/12930542/d6d14aa3-image.png)

## Where to find this information:

The API key, API secret key, and your Workspace ID can all be created and found by accessing the API CREDENTIALS section: [https://sendspark.com/settings/api-credentials](https://sendspark.com/settings/api-credentials)

![Sendspark API Credentials section showing keys and Workspace ID](/images/12930542/548e6a15-image.png)
