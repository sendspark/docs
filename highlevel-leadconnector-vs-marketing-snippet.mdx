title: "HighLevel: LeadConnector vs Marketing Snippet"
description: "Learn how to choose between LeadConnector and Marketing Snippet code in HighLevel for video campaigns and email automations."
---

Like many email platforms, GoHighLevel offers the option to send emails in different ways. One is the plain text option, and another is the HTML builder option.

When you generate a dynamic video campaign with Sendspark, you'll have or you'll see two options in order to share videos in GHL: "LeadConnector" and "LeadConnector Marketing". Both are valid and both work as long as you use them in the correct place:

![Lead Connector](/images/12822471/f3bffc54-image.png)

## In an Automation

If you are inside an automation and you plan on adding an email step, and you want to send a video there, Yyou can simply click the "Delete Connector" option in order to copy the snippet code, and you can simply paste it in the body of the email.

![Draft Email](/images/12822471/f946f06d-image.png)

## When building an email with HTML tools

Go High Level also offers the option to build emails with blogs by simply dragging and dropping the elements inside of the email preview.

In this instance, you'll need the "Lead Connector Marketing" code and once you copy it, you need to make sure that you are adding it into the code block as shown below.

![Code Block](/images/12822471/e446c3b7-image.png)

You'll notice that you can grab the text element as well and paste it there and although it might look like it's fine, it won't work when you send the email. So be sure to use the code block and drop it there.

![Paste the Code](/images/12822471/eb444eff-image.png)
