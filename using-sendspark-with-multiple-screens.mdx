If you use multiple monitors, you may want to control which screen Sendspark opens on, especially since the camera bubble can only appear on the screen where the application is launched.

## Opening Sendspark on a Different Screen

By default, Windows applications, including Sendspark, open on the monitor set as the primary display. To open Sendspark on a different screen:

- **Change the Primary Display** — Right-click on the desktop and select Display settings. In the list of connected monitors, click on the one you want to set as your main display. Check the box for "Make this my main display."
- **Restart Sendspark** — Fully close and exit Sendspark, then reopen it to ensure it launches on the new main display.

## Multi-Screen FAQs

- **Can I move the camera bubble to another screen during a session?** — Unfortunately, no. The Sendspark camera bubble can only remain on the monitor where the app was initially opened.
- **Why does Sendspark always open on the same screen?** — Sendspark, like other applications, opens on the screen set as the main display in your system settings.
- **How can I record content from a different screen?** — When selecting what to share (step 6), you can choose content from any screen, even if the camera bubble is fixed to the primary screen.

## Troubleshooting Tips

- Ensure that the primary display is correctly set under your system's Display Settings.
- Fully close Sendspark before switching display settings for the changes to take effect.
