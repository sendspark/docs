title: "LeadConnector (GHL) Agentic Workflow Automation"
description: "Connect Sendspark with LeadConnector (GHL) to automate dynamic video sending in agentic workflows for scalable personalized outreach."
---

Now with the Agentic Workflow, you're able to create a video easily without having to handle webhooks.

By using the Trigger :  "Tag Update" on a contact, you're able to start any action in Sendspark:

![](/images/14463226/137d0558-image.png)

Simply create a new automation, look for lead connector, contact tag update

Click on the filtering option:

![](/images/14463226/4350c13b-image.png)

Click on the first field and type "tag" and click on TAGS to add the initial statement of the condition.

![](/images/14463226/2ad9a182-image.png)

Condition type can be "equals":

![](/images/14463226/feb9a6d5-image.png)

And simply type the tag you'll use in GHL:

![](/images/14463226/1f5a1b48-image.png)

Now every time you add this tag into a contact,Then anything else will happen afterward, such as generating a video in Sendspark.
