With Sendspark Agentic Workflows, you can trigger a video from GoHighLevel (LeadConnector) without handling webhooks. Create a new automation, choose the LeadConnector "Contact Tag Update" trigger, add a filter where the tag equals the tag you use in GHL, and every contact given that tag automatically starts an action in Sendspark, such as generating a video.

## Key takeaways

- The Agentic Workflow lets you create a video from GoHighLevel easily, without having to set up webhooks.
- Use the LeadConnector **Contact Tag Update** trigger to start any Sendspark action.
- Configure the filter by clicking the first field, typing "tag" and selecting **TAGS**, setting the condition type to **equals**, and entering the exact tag you'll use in GHL.
- Once set up, applying that tag to a contact automatically runs the following steps, such as generating a video in Sendspark.

Now with the Agentic Workflow, you're able to create a video easily without having to handle webhooks.
