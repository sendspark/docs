## 🛠️ HubSpot Warning: “This HTML module can contain HubL code” – What to Do

If you're seeing a warning like this when building an email in **HubSpot Marketing Hub**:

“This HTML module can contain HubL code.”

Don’t worry — this is a **false warning** caused by HubSpot's default behavior. **Sendspark snippets do *not* contain any HubL code.**

## 📌 When Does This Happen?

This usually appears after you:

1. Use the **drag-and-drop email builder** in HubSpot.
2. Paste your **Sendspark dynamic video snippet** into a custom HTML block.
3. Exit the email editor and come back later — the warning may show up.

## ✅ How to Fix It

To clear the message:

1. **Remove the existing HTML block** that shows the warning.
2. Go back to your Sendspark campaign and **copy the dynamic video snippet** again.
3. Add a **new HTML block** in the email.
4. Paste the snippet into the new block.
5. Click **“Review and Send”** — the warning should be gone.

✅ Pro Tip: Always preview your email before sending to ensure the video GIF and fallback image display correctly.
