To connect Sendspark with PipeDrive, open PipeDrive's Web Visitors under Leads and copy the tracker ID shown in brackets (for example, DzLR5a5WNoY7BoQ2). Then paste it into the PipeDrive ID field on your [Sendspark integrations page](https://sendspark.com/settings/integrations). This sends Sendspark page visits to PipeDrive so you can track engagement and visualize it in dashboards.

## Key takeaways

- The integration sends Sendspark video-page visits to PipeDrive so you can track contact engagement and visualize the data in dashboards.
- You connect the two tools by copying your PipeDrive Web Visitors tracker ID into the PipeDrive ID field in Sendspark's integration settings.
- The tracker ID lives under **Leads → Web Visitors** in PipeDrive. Copy only the code between the brackets marked in red, and nothing else.
- Setup is done in Sendspark's [Integrations settings](https://sendspark.com/settings/integrations); no coding is required.

The Sendspark for PipeDrive integration allows you to easily measure the impact video is having on your business.

This integration will allow you to send Sendspark page visits to PipeDrive to track engagement with contacts and visualize video engagement data in dashboards.

---

1) Access your PipeDrive website

2) Head to "Leads" on the left hand side corner:

![PipeDrive home page](/images/8843739/a1b22825-image.png)

3) If you are new to PipeDrive, click on "Start a free trial", otherwise access Web Visitors.

4) You should now be able to see the following page. Simply COPY the last section you see between brackets and marked in red, and nothing else.

In this example: DzLR5a5WNoY7BoQ2

![Tracker ID page](/images/8843739/a7177139-image.png)

5) Go back to Sendspark integrations section: https://sendspark.com/settings/integrations  
  
6) Paste it in the PipeDrive ID:

![Integrations page](/images/8843739/9b08ce3f-image.png)

7) You are all set!

## Frequently asked questions

<AccordionGroup>
  <Accordion title="What does the Sendspark for PipeDrive integration do?">
    It sends your Sendspark video-page visits to PipeDrive, so you can track engagement with contacts and visualize video engagement data in PipeDrive dashboards.
  </Accordion>
  <Accordion title="Where do I find my PipeDrive tracker ID?">
    In PipeDrive, go to **Leads** and open **Web Visitors**. If you are new to PipeDrive, start the free trial first. Copy only the last section shown between the brackets and marked in red (for example, DzLR5a5WNoY7BoQ2), and nothing else.
  </Accordion>
  <Accordion title="Where do I paste the tracker ID in Sendspark?">
    Go to your Sendspark integrations section at https://sendspark.com/settings/integrations and paste the ID into the PipeDrive ID field. Once saved, you're all set.
  </Accordion>
  <Accordion title="Do I need a PipeDrive account to use this?">
    Yes. You need access to PipeDrive's Web Visitors feature to get a tracker ID. If you don't have an account yet, you can start a free trial from the Leads page.
