✅ Supported Facebook Pages

Sendspark can successfully use backgrounds from public Facebook pages — specifically:

- **Non-private personal profile pages** (usually start with `/people/`)
- **Business pages** (often start with `/p/` or a simple username like `/woodscoffee`)

💡 Examples of working pages:

- [https://www.facebook.com/people/Patriot-Remodeling/100032747787102/](https://www.facebook.com/people/Patriot-Remodeling/100032747787102/)
- [https://www.facebook.com/woodscoffee](https://www.facebook.com/woodscoffee)
- [https://www.facebook.com/p/FD-Construction-Sioux-Falls-100093578633502/](https://www.facebook.com/p/FD-Construction-Sioux-Falls-100093578633502/)

🚫 Unsupported Pages: Facebook restricts access to certain types of pages. The following **will not work** as backgrounds in Sendspark:

- Facebook **Marketplace** listings
- **Ads** pages or sponsored content
- **Timeline** or **personal feed** links
- **Reels** URLs
- Any **private or restricted** content

These pages are either protected by Facebook’s privacy settings or require authentication, making them inaccessible for background rendering.

⚠️ Possible Errors You Might See: If the page you’re using is not public or has been removed, you may see one of these error messages:

- **"Page is content restricted. Please try a different page.”**

→ This means the page is private or region-locked.Try using another Facebook link that matches the supported formats above.
