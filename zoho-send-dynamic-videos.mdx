## How to Share a Dynamic Video Campaign through Zoho

## 1. Create your Dynamic Video Campaign

- Generate your Dynamic Videos and ensure all of them have been processed ![Sendspark dynamic video campaign dashboard showing processed videos list](/images/8392439/b5743626-image.png)

## 2. Download the updated CSV File

![Download CSV button highlighted in Sendspark campaign settings panel](/images/8392439/2bc3b4fa-image.png)

- Download the CSV file of leads to get the updated version where any unsuccessful videos are filtered out.
- Click on **Download CSV** to get the updated version.

## 3. Access Zoho Account

- Log in to your Zoho account.
- Go to **Contacts** and click on **Manage Lists**. ![Zoho Contacts page with Manage Lists option visible in navigation](/images/8392439/bb564202-image.png)

## 4. Create and Import Contact List

- Create a new list (e.g., "July List"). ![Zoho create new contact list dialog with name input field](/images/8392439/982b28fb-image.png)
- Click on **Import Contacts**. ![Zoho Import Contacts button in list management interface](/images/8392439/8dde1285-image.png)
- Upload the downloaded CSV file: ![CSV file upload interface in Zoho with file selection dialog](/images/8392439/5dc0f875-Jul-12-2024_4-42-28_PM.gif)
- Map only the mandatory fields (email and first name: ). Optionally, map job title and company name.
- Save and import the contacts.

Note: You don't need to map Video Link or background URL.

## 5. Create an Email Campaign

- Go to **Campaign** and select **Create Campaign** &gt; **Email Campaign** &gt; **Regular Email**.
- Name your campaign (e.g., "July Campaign").
- Add a subject line and sender information.

![Zoho email campaign creation form with campaign name and subject fields](/images/8392439/1824a90f-image.png)

## 6. Add Recipients

- Click on **List** and select your newly imported contact list.

## 7. Create Email Content

- Choose **Pre-designed Templates**.
- Use the **Insert HTML** tool from the left sidebar.
- Drag and drop the HTML block into your email content.

## 8. Insert Dynamic Video Campaign

- Go back to your dynamic video campaign and select **Share Campaign** &gt; **Zoho**.
- Copy the provided HTML code and paste it into the HTML block in Zoho.
- Apply the changes.

![Sendspark Share Campaign menu with Zoho option and HTML code display](/images/8392439/7a2aa435-eeee.gif)

## Side Notes

- **Variables and Placeholders:** During test emails, placeholders might not display correctly due to variable placeholders. For accurate testing, add yourself as a contact and send the campaign to a secondary email with a different domain.
- **Spam Issues:** Sending emails from the same inbox to the same inbox may be flagged as spam. Use a different domain for test emails if needed.
